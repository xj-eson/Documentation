**用户语言**
==============

**请求地址**

::

   GET https://openapi.ctcfile.com/v1/users/languages

**返回值**

=========================== ====== ====================================
属性                        类型   说明
=========================== ====== ====================================
id                          int    ID
name                        string 名称
value                       string 语言标识值
english                     string 英语名
native                      string 本地名
=========================== ====== ====================================
