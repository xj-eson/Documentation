**用户的领域类型**
====================

**请求地址**

::

   GET https://openapi.ctcfile.com/v1/users/domain-types

**返回值**

=========================== ====== ====================================
属性                        类型   说明
=========================== ====== ====================================
id                          int    ID
name                        string 领域名称
=========================== ====== ====================================

**返回数据示例**

正常返回

.. code:: json

    [
        {
            "id": 6,
            "name": "通用版"
        }
    ]

错误时返回
