**获取文件页数**
==================================

**请求地址**

::

   POST https://openapi.ctcfile.com/v1/files/pages

**请求参数**

==== ====== ====== ==== ============
属性 类型   默认值 必填 说明
==== ====== ====== ==== ============
data object        是   这里传空对象`{}`。**注:该参数必须为第一个**
file binary        是   转换文件(流)
==== ====== ====== ==== ============

**返回值**

====== ======
类型   说明
====== ======
int    文件页数
====== ======


错误时返回

   :doc:`../overview/StatusCode`
