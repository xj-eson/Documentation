**office文档、图片转pdf文档**
==================================

**请求地址**

::

   POST https://openapi.ctcfile.com/v1/conversions/file-to-pdf

**请求参数**

==== ====== ====== ==== ============
属性 类型   默认值 必填 说明
==== ====== ====== ==== ============
data object        是   这里传空对象`{}`。**注:该参数必须为第一个**
file binary        是   转换文件(流)
==== ====== ====== ==== ============

**返回值**

====== ======
类型   说明
====== ======
stream 文件流
====== ======

*Response Heads(响应头信息)*

================= ====== ============================================================
属性              类型   说明
================= ====== ============================================================
x-user-totalpages int    剩余的总页数
x-user-pages      int    当前使用的页数
================= ====== ============================================================
